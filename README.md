# SENDIT

Always-on social crash demo for publisher outreach.  
**Don't paper. Peel half. Chase more.**

## Start here
1. Read **`TRACKER.md`** — status of packaging tasks 1–9  
2. Fill **`CONTACT.md`** before sending email  
3. Public demo URL lives in **`DEMO.md`**

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

Deploy: GitHub Pages via `.github/workflows/deploy-pages.yml` on push to `main`.
