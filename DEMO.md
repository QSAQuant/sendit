## Controls (for BD playtests)
- Tabs: **PLAY · TABLES · MYTHS · TAPE · VERIFY · EV LAB**
- **SEND** — join next fuse (Space); queues if mid-round
- **PAPER** OFF / HALF / FULL — bust cover on notional
- **HEAT 1× / 2×** — capped exposure (H); 3% fee on extra
- **SNAP** DUST/CHOP/COOK/MOON — side bet on death band
- **ASH** — 10% of margin lost → next fuse only
- **PEEL 50%** (P) · **CASH** voids PAPER (Space while riding)
- Loadout code under quote — copy via EV LAB
- Lanes **1 / 2 / 3** = SLOW / SEND / DEGEN

## Fairness strip
- Pre-round: commitment **hash** shown
- Post-snap: **seed** revealed; crash derived from seed + lane house edge
